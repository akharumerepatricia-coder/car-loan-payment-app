# 🚗 Car Loan Tracker — Canada

A Canadian car loan simulator and personal payment tracker built with React + Vite.

## Features
- Simulate monthly & biweekly payments with real Big 5 bank rates
- Customize vehicle price, down payment, loan term, and interest rate
- Personal tracker tab: log real payments, see principal vs interest split, donut pie chart
- Compare all 5 banks side by side
- Extra payment modeling with interest savings calculator

## Deploy to Vercel (no terminal needed)
1. Upload this folder to a GitHub repo
2. Go to vercel.com → Import that repo
3. Click Deploy — done ✅

## Local development
```bash
npm install
npm run dev
```

## Tech stack
- React 18
- Vite 5
- Pure SVG charts (no chart library dependencies)
- Zero external UI dependencies
